Цель проекта

Этот проект демонстрирует реализацию и сравнение различных брокеров сообщений в едином Spring Boot приложении.

Цели обучения:

  • Понимание различных паттернов обмена сообщениями
  • Сравнение характеристик производительности брокеров
  • Изучение интеграции Spring Boot с системами обмена сообщениями
  • Реализация готовых к production мониторинга и проверок здоровья
Портфолио проект

Этот проект демонстрирует современные практики Java разработки и паттерны корпоративной интеграции.

Продемонстрированные навыки:

  • Микросервисная архитектура
  • Интеграция брокеров сообщений
  • Проектирование REST API
  • Веб-приложения реального времени
  • Тестирование производительности
  • Production мониторинг
